# 在线答题考试系统

基于Vue.js 3开发的在线答题考试系统，支持题库管理、试卷创建、在线考试等功能。

## 功能特性

### 🎯 核心功能
- **题库管理**: 支持单选题和多选题的添加、删除和管理
- **试卷创建**: 灵活的试卷创建，可选择题目和设置考试参数
- **在线考试**: 完整的考试流程，包含计时器和实时答题
- **计分系统**: 支持加分制和减分制两种计分模式
- **结果统计**: 考试结果展示和统计分析

### 📊 计分模式
1. **加分制**: 答对题目获得相应分数，答错不扣分
2. **减分制**: 答对题目获得相应分数，答错扣除相应分数（最低分为0）

### 🎨 界面特色
- 现代化的响应式设计
- 直观的用户界面
- 流畅的交互体验
- 移动端友好

## 技术栈

- **前端框架**: Vue.js 3 (Composition API)
- **构建工具**: Vite
- **样式框架**: Tailwind CSS
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **图标**: Heroicons

## 快速开始

### 方式一：使用Vue项目（推荐）

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 访问应用：
打开浏览器访问 `http://localhost:5173`

### 方式二：使用单文件版本

1. 启动HTTP服务器：
```bash
python3 server.py
```

2. 访问应用：
打开浏览器访问 `http://localhost:9000/test.html`

## 使用指南

### 1. 题库管理
- 点击"题库管理"进入题库页面
- 点击"添加题目"创建新题目
- 支持单选题和多选题两种类型
- 可设置题目分值和正确答案

### 2. 试卷创建
- 点击"试卷管理"进入试卷页面
- 点击"创建试卷"开始创建
- 设置试卷标题、描述和考试时长
- 选择计分模式（加分制/减分制）
- 从题库中选择题目

### 3. 在线考试
- 在首页或试卷列表中点击"开始考试"
- 查看考试说明和参数
- 点击"开始考试"进入答题界面
- 在规定时间内完成所有题目
- 提交试卷查看结果

## 开发说明

### 添加新功能
1. 在相应的Vue组件中添加UI
2. 在Pinia store中添加状态管理逻辑
3. 更新路由配置（如需要）

### 自定义样式
项目使用Tailwind CSS，可以通过修改`tailwind.config.js`来自定义主题。

### 数据持久化
当前版本使用内存存储，刷新页面后数据会丢失。如需持久化，可以：
1. 使用localStorage存储数据
2. 集成后端API
3. 使用数据库存储
